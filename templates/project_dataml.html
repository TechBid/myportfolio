Shiny Applications Suite

A comprehensive collection of 8 interactive R Shiny applications for data analysis, visualization, and decision-making.

This suite demonstrates the power of R Shiny for building full-stack data science applications. It includes tools for business analytics, statistical simulation, financial forecasting, geospatial analysis, and quality control. All apps feature a consistent, modern UI using bslib and are deployed for real-time interaction.

All 8 Applications

  • 1. Business Analytics Dashboard: Track sales, KPIs, and customer churn with interactive visualizations.
  • 2. Data Explorer: Upload and explore datasets with dynamic visualizations and filtering capabilities.
  • 3. Statistical Lab: Interactive tools for A/B testing, confidence intervals, regression analysis, and distribution visualization.
  • 4. Model Deployment Hub: Deploy and test machine learning models (Random Forest, Linear Regression) with interactive predictions.
  • 5. Financial Analysis Suite: Portfolio optimization, ARIMA/ETS forecasting, and Value at Risk (VaR) analysis.
  • 6. Geospatial Explorer: Interactive maps of Kenya with clustering, heatmaps, and choropleth visualizations.
  • 7. Quality Control Monitor: Real-time SPC charts, process capability analysis (Cp, Cpk), and manufacturing quality monitoring.
  • 8. Survey Analyzer: Analyze survey data with Net Promoter Score (NPS), satisfaction metrics, and downloadable reports.
